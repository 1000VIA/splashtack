Introducción

JavaScript es el lenguaje de programación más popular del mundo.

Javascript fue creado por Brendan Eich en 1995 y se convirtió en un estándar ECMA en 1997.

Javascript y Java son lenguajes completamente diferentes, tanto en concepto como en diseño.

ECMA-262 es el nombre oficial del estándar. ECMAScript es el nombre oficial del idioma.

JavaScript te permite cambiar el contenido de los elementos (etiquetas) HTML.


Ejemplo:

<p id="demo">Cámbiame</p>

<button type="button" onclick='document.getElementById("demo").innerHTML = "Hola Javascript!"'>Haz clic sobre mí</button>

Resultado:

Cámbiame


Con JavaScript puedes cambiar el valor de los atributos de las etiquetas de HTML.


Ejemplo:

<button onclick="document.getElementById('miImagen').src='./images/pic_bulbon.gif'">Enciende la luz</button>

<img id="miImagen" src="./images/pic_bulboff.gif" style="width: 100px; height: 150px;">

<button onclick="document.getElementById('miImagen').src='./images/pic_bulboff.gif'">Apaga la luz</button>

Resultado:

En este caso JavaScript cambia el valor del atributo src (source) de una imagen.


Javascript puede cambiar los estilos de HTML


Ejemplo:

<p id="demo1">Texto</p>

<button type="button" onclick="document.getElementById('demo1').style.fontSize='35px'">Agrandar Texto</button>

Resultado:

Texto


Con JavaScript puedes ocultar los elementos de HTML.


Ejemplo:

<p id="demo2">JavaScript puede ocultar los elementos de HTML.</p>

<button type="button" onclick="document.getElementById('demo2').style.display='none'">Ocultar texto</button>

Resultado:

JavaScript puede ocultar los elementos de HTML.


Con JavaScript puedes ocultar los elementos de HTML.


Ejemplo:

<p>Javascript te permite mostrar elementos ocultos de HTML.</p>

<p id="demo3" style="display:none">Hello JavaScript!</p>

<button type="button" onclick="document.getElementById('demo3').style.display='block'">Mostrar elemento</button>

Resultado:

Javascript te permite mostrar elementos ocultos de HTML.

Hello JavaScript!

Ubicación de JavaScript

Todo el código de JavaScript va dentro de las etiquetas <script></script>.

Puedes colocar las etiquetas <script></script> dentro de las etiquetas <body></body> y <head></head>.

Para mejorar la velocidad de carga de tu página web, se recomienda colocar las etiquetas <script></script> antes de la etiqueta de cierre </body>.


Ejemplo:

<!DOCTYPE html>
<html>
<body>

    <p id="demo5"></p>

    <script>
        document.getElementById("demo5").innerHTML = "Mi primer código con JavaScript.";
    </script>

</body>
</html>

Resultado:

Usando innerHTML

El atributo id define el elemento HTML. La propiedad innerHTML define el contenido de HTML.
