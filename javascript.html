El poder dinámico de JS

Con JavaScript puedes cambiar el contenido de las etiquetas HTML.

Ejemplo:

<p id="demo">Cámbiame</p>

<button type="button" onclick='document.getElementById("demo").innerHTML = "Hola Javascript!"'>Haz clic sobre mí</button>

Resultado:

Cámbiame
