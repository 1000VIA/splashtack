Introducción

HTML es el lenguaje de marcado estándar para crear páginas web.

Encabezados

Los encabezados se utilizan para crear los títulos de una página web.

Ejemplo:

<h1>Encabezado h1</h1>

<h2>Encabezado h2</h2>

<h3>Encabezado h3</h3>

<h4>Encabezado h4</h4>

<h5>Encabezado h5</h5>
<h6>Encabezado h6</h6>

Resultado:

Encabezado h1

Encabezado h2

Encabezado h3

Encabezado h4

Encabezado h5
Encabezado h6

Párrafos

Un párrafo siempre empieza en una línea nueva y usualmente es un bloque de texto.

Ejemplo:

<p>Este es un párrafo</p>

<p>Este es otro párrafo</p>

Resultado:

Este es un párrafo

Este es otro párrafo

Estilos

El atributo style se utiliza para añadir estilos a un elemento: El color, el tipo de fuente, el tamaño del texto y más.

Ejemplo:

<p>Soy un párrafo normal</p>

<p style="color: red">Soy un párrafo rojo</p>

<p style="font-size: 50px">Soy un párrafo grande</p>

Resultado:

Soy un párrafo normal

Soy un párrafo rojo

Soy un párrafo grande

Formato de Texto

HTML cuenta con varios elementos para darle formato al texto.

Ejemplo:

<p>Texto normal<p>
<p><b>Texto en negrita</b></p>
<p><strong>Texto importante</strong></p>
<p><em>Texto enfatizado</em></p>
<p><mark>Texto resaltado</mark></p>
<p><small>Texto pequeño</small></p>
<p><del>Texto eliminado </del></p>
<p><ins>Texto insertado</ins></p>
<p><sub>Texto de subíndice</sub></p>
<p><sup>Texto de superíndice</sup></p>

Resultado:

Texto normal

Texto en negrita

Texto importante

Texto enfatizado

Texto resaltado

Texto pequeño

Texto eliminado

Texto insertado

Texto de subíndice

Texto de superíndice
