Introducción

HTML es el lenguaje de marcado estándar para crear páginas web.

Encabezados

Los encabezados se utilizan para crear los títulos de una página web.

Ejemplo:

<h1>Encabezado h1</h1>

<h2>Encabezado h2</h2>

<h3>Encabezado h3</h3>

<h4>Encabezado h4</h4>

<h5>Encabezado h5</h5>
<h6>Encabezado h6</h6>

Resultado:

Encabezado h1

Encabezado h2

Encabezado h3

Encabezado h4

Encabezado h5
Encabezado h6

Párrafos

Un párrafo siempre empieza en una línea nueva y usualmente es un bloque de texto.

Ejemplo:

<p>Este es un párrafo</p>

<p>Este es otro párrafo</p>

Resultado:

Este es un párrafo

Este es otro párrafo

Estilos

El atributo style se utiliza para añadir estilos a un elemento: El color, el tipo de fuente, el tamaño del texto y más.

Ejemplo:

<p>Soy un párrafo normal</p>

<p style="color: red">Soy un párrafo rojo</p>

<p style="font-size: 50px">Soy un párrafo grande</p>

Resultado:

Soy un párrafo normal

Soy un párrafo rojo

Soy un párrafo grande

CSS

CSS significa hojas de estilo en cascada, estas hojas de estilos controlan el diseño de todas las páginas web.

Con CSS puedes controlar el color, la fuente, el tamaño del texto, el espaciado entre los elementos, la posición de los elementos, las imágenes de fondo, los colores de fondo, y todo lo relacionado con el aspecto y el diseño de una página web.

Ejemplo:

<!DOCTYPE html>
<html lang="es">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>CSS</title>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<div class="elemento1">Elemento 1</div>
<div class="elemento2">Elemento 2</div>
<div class="elemento3">Elemento 3</div>
<div class="elemento4">Elemento 4</div>
<div class="elemento4">Elemento 5</div>
</body>
</html>
.elemento1{
    color: red;
}

.elemento2{
    color: #fff;
    background: blue;
    padding: 10px;
    border-radius: 5px;
    width: 150px;
}

.elemento3{
    color: violet;
}

.elemento4{
    font-size: 50px;
}

.elemento5{
    text-decoration: line-through;
}

Resultado:

elemento 1
elemento 2
elemento 3
elemento 4
elemento 5

Enlaces

Los enlaces se encuentran en casi todas las páginas web. Un enlace no tiene que ser texto, también puede ser una imagen o cualquer otro elemento HTML.

Ejemplo:

<a href="https:/www.google.com/" class="enlace">Enlace normal</a>
<a href="https:/www.google.com/" target="_self">Enlace con atributo _self</a>
<a href="https:/www.google.com/" target="_blank">Enlace con atributo _blank</a>
<a href="https:/www.google.com/" target="_parent">Enlace con atributo _parent</a>
<a href="https:/www.google.com/" target="_top">Enlace con atributo _top</a>

Resultado:

Enlace normal Enlace con atributo _self Enlace con atributo _blank Enlace con atributo _parent Enlace con atributo _top
